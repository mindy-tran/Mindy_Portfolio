Project 8: Alien Jump - Game Project Management

Link to play Alien Jump

Link to GitHub Repository

Imagine you are an alien, and after an emergency crash landing onto Earth, you must get back to the mothership to return home.

This game is inspired by the Doodle Jump mobile game that has a player jumping from platforms and getting vertically higher. There will be static platforms, moving platforms, and enemies that try to make your player fall. Unlike the mobile game, there will be 3 different levels with differing environments starting with an underground level, then an above-ground level, and a level in space. The storyline for the game is based on the idea that an alien has crashed onto Earth and is escaping into space to return home.

  • Designed and developed a 3D web game in C# leveraging agile methodologies, conducting playtest sessions with over 30 users, and making iterative improvements to the gameplay and game progression
  • Managed a 4-person team through a multi-build development cycle (pre-alpha, alpha, beta, final), GitHub version control, Kanban task tracking, and issue tracking
  • Ultimately responsible for the team’s deliverables, keeping the project on track, and making executive decisions
